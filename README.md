# 🎓 Zephyra — Система управління навчальним процесом

**Zephyra** — це сучасна, потужна та гнучка система для автоматизації академічного обліку, розроблена на базі **Django 5.2** та **MySQL**. Система забезпечує повний цикл контролю успішності: від планування розкладу до формування складних аналітичних звітів з використанням зважених коефіцієнтів.

---

## 🚀 Основні Можливості

### 👨‍💼 Адміністратор (Керування та Контроль)
- **Управління користувачами**: CRUD-операції для студентів, викладачів та адміністраторів з хешуванням паролів.
- **Глобальні довідники**: Повне керування навчальними групами та предметами.
- **Навчальне навантаження**: Призначення викладачів на конкретні дисципліни для певних груп.
- **Інтерактивний розклад**: Налаштування тижневого розкладу занять.
- **Потужна аналітика**: Генерація звітів про пропуски та академічний рейтинг.

### 👨‍🏫 Викладач (Журнал та Оцінювання)
- **Персональний журнал**: Зручне введення оцінок та фіксація пропусків для закріплених груп.
- **Зважена система оцінювання**: Можливість налаштування "ваги" для різних типів занять (напр. Екзамен — 50%, Квіз — 10%). Рейтинг студента вираховується автоматично на основі цих коефіцієнтів.
- **Управління темами**: Фіксація тем проведених занять та типів уроків.

### 👨‍🎓 Студент (Прозорість та Статистика)
- **Особистий залік**: Перегляд усіх отриманих оцінок з деталізацією по предметах та коментарями викладачів.
- **Облік відвідуваності**: Перегляд пропусків (з виділенням поважних та неповажних причин).
- **Просунуті фільтри**: Студенти можуть фільтрувати свої дані за періодом, предметом, балом та пошуковим запитом.

## 🛠️ Технологічний Стек

- **Backend**: Python 3.14+ / Django 5.2.7
- **Database**: MySQL (через `mysql-connector-python`)
- **Frontend**: Vanilla CSS (Premium Glassmorphism Design), JavaScript (AJAX/Fetch API)
- **Reporting**: CSV Export, Bayesian Weighted Rating

## 📦 Встановлення та Налаштування

### 1. Підготовка середовища
```bash
git clone <url_репозиторію>
cd Django
python -m venv .venv
source .venv/bin/activate  # Для Windows: .venv\Scripts\activate
pip install -r requirements.txt
```
*(Примітка: якщо `requirements.txt` відсутній, встановіть основні пакети: `pip install django mysql-connector-python python-dotenv`)*

### 2. Налаштування бази даних
Створіть файл `.env` у корені проекту та додайте параметри підключення до MySQL:
```env
DB_NAME=edutrack_db
DB_USER=root
DB_PASSWORD=ваш_пароль
DB_HOST=localhost
DB_PORT=3306
SECRET_KEY=ваш_секретний_ключ
```

### 3. Міграції та старт
```bash
python manage.py migrate
python manage.py runserver
```

## 📊 Аналітичні Звіти

Система Zephyra використовує складні алгоритми для об’єктивної оцінки успішності:

- **Рейтинг (Bayesian Average)**: Вираховується не як просте середнє арифметичне, а як зважений рейтинг. Він враховує вагу кожного типу заняття та загальну кількість оцінок, що робить його стійким до "випадкових" високих балів.
- **Звіт про пропуски**: Детальна статистика відвідуваності з можливістю фільтрації за предметом, групою та періодом.

## 📂 Структура Проекту

- `main/` — Основний додаток (моделі, бізнес-логіка, шаблони).
- `edutrack_project/` — Налаштування проекту та маршрутизація.

## 🛡️ Безпека
- Всі паролі зберігаються у вигляді захищених хешів (PBKDF2).
- Рівневий доступ до функцій (Admin / Teacher / Student) через декоратори `role_required`.
- Захист від SQL Injection через використання Django ORM.
